# design.md

## 1. デザインコンセプト
### **Digital Crafts（デジタルな工芸品）**
懐かしさを感じるレトロポップな色使いと、最新のニューモーフィズム（柔らかい陰影による立体感）を組み合わせた、温かみのあるモダンなデザイン。

## 2. カラーパレット（レトロポップ）
* **Base Background**: `#E8E2E4`（温かみのあるモーヴグレー）。
* **Primary Color (Done)**: 
    * マスタードイエロー: `#E3B04B`
    * セージグリーン: `#8A9A5B`
    * テラコッタ: `#C05555`
* **Accent (Text/Icon)**: 
    * ヴィンテージ・ネイビー: `#2C3E50`

## 3. UIスタイル：ニューモーフィズムの適用
* **カードの状態 (Undone / 凸)**: 
    * ライトシャドウ（左上）とダークシャドウ（右下）を落とし、画面から浮き上がっているように見せる。
* **カードの状態 (Done / 凹)**: 
    * 内側にインナーシャドウを入れ、ボタンが押し込まれたように表現。
    * 背景色がレトロカラーに染まり、物理的なスイッチがONになったような満足感を与える。

## 4. インタラクション
* **Haptic**: タップ時に物理スイッチを模した「カチッ」という触覚フィードバック（Haptic Feedback）を返す。
* **Animation**: 凸から凹への変化は、物理的なバネ感を感じさせるスムーズなアニメーション（0.2s）で実装。
